# JavaScript Mini Projects Collection

This repository contains 4 mini projects built using **HTML**, **CSS**, and **JavaScript** to revise core JavaScript concepts and strengthen frontend development skills.

## Projects Included

### 1. Color Changer 🎨

A simple project that allows users to change the background color by clicking different color buttons.

**Concepts Practiced:**

* DOM Manipulation
* Event Listeners
* CSS Styling with JavaScript

---

### 2. BMI Calculator 📊

A BMI calculator that takes height and weight as input and calculates the Body Mass Index.

**Concepts Practiced:**

* Form Handling
* User Input Validation
* Conditional Statements
* DOM Updates

---

### 3. Digital Clock ⏰

A real-time digital clock displaying the current time.

**Concepts Practiced:**

* Date Object
* setInterval()
* Dynamic Content Updates

---

### 4. Guess The Number 🎯

A number guessing game where users try to guess a randomly generated number within limited attempts.

**Concepts Practiced:**

* Random Number Generation
* Loops and Conditions
* User Interaction
* Game Logic

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Learning Outcome

These projects were built as part of my JavaScript revision journey to reinforce fundamental concepts through practical implementation. Working on hands-on projects helped me gain a better understanding of JavaScript and improve problem-solving skills.
